# Natik Hospital Management System

A hospital management system for managing patients, doctors, appointments, and more.

## Getting Started

Clone the repository and follow the setup instructions below.

```bash
git clone <repo-url>
cd Natik_Hospital
```

## Contributing

1. Fork the repository or request access
2. Create a feature branch from `dev`: `git checkout -b feature/your-feature dev`
3. Commit your changes with clear messages
4. Open a pull request targeting the `dev` branch

## Branch Strategy

- `main` — stable, production-ready code
- `dev` — active development, merge features here first
